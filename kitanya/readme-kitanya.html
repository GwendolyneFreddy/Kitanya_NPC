Components of the Mod

 

You may install any of these components individually or together.  They have all been tested with each other:

 

Component 1:  Kitanya Conversation/Romance

 

This component is the NPC herself.  It includes all of the new dialogues, sounds, spells, items, etc.

 

Component 2:  Wizard Slayer Bugs Fix

 

Bioware made a mistake when coding the "Protection from..." scrolls that are found throughout Baldur's Gate II.  Although the descriptions say that wizard slayers cannot use these scrolls, they actually can.  This bug is not fixed by either the official patches or by Baldurdash.  Installing this component will fix the problem.

 

After installing this component, wizard slayers will no longer be able to use these scrolls (as it should be).

 

Component 3:  Flaming Sword for Kitanya

 

 1 Pixel Productions introduced new graphics for a flaming sword several months ago.  It applied the flaming graphics to swords included with the original game, but not mods.  This component will add these new effects to Kitanya’s flaming sword.  Please note that this component requires that 1PPv3.0 Flaming Swords be installed.

 

How to Install

 

This version of Kitanya is not compatible with previous versions.  If you have version 6.2.1 or earlier installed then you must uninstall it before installing version 6.2.2

 

The Kitanya NPC Conversation and Romance mod is distributed as a self-extracting install program (or as a ZIP archive for Mac OS X users - See Mac OSX Specific Instructions).  To install the mod, first run the installer after it has finished downloading.  You will have the option to choose where to install the mod.  The installer will search your registry and display the default installation path but if you have multiple installations of BG2, you may wish to change this.  The path must be the root directory of whatever game installation you wish to install the mod onto (this same path must contain chitin.key and dialog.tlk).  You may then click install.  The installer will automatically extract all needed mod resources and will launch the WeiDU executable for you.  Answer the questions that are presented to you as they appear to choose which components of the mod that you wish to install.

 

Several DOS windows may open up during the installation.  You should allow these windows to remain open and continue what they are doing.  Due to the need to prevent the download from being several hundred MB in size, all of the sound files have been heavily compressed.  They are decompressed into a format that the game can use immediately after the mod is installed.  These DOS windows may appear as a result of this decompression.

 

Mac OSX Specific Instructions:  The Kitanya mod is not compatible with the Mod Organizer.  If you have a previous version of Kitanya installed, it is necessary for you to uninstall it before installing the current version.  Please note that this means that you must remove any previous version of Kitanya before removing any files from the ZIP archive.

 

Once all files have been extracted to your root BG2 directory, double-click on setup-kitanya.command.  From here, follow the same instructions as the PC users.

 

 

Uninstallation Instructions

 

Run the setup-kitanya executable again.  It will give you the option to uninstall.  This step is the same for PC and Mac users.

 

 

Compatibility Concerns

 

This mod should be compatible with all known WeiDU mods.  There should not be any issues if Kitanya is installed last.  I have tested it with, and verified compatibility with the following mods:

 

- Kelsey SoA & ToB

- Solaufein (only if Solaufein is installed first)

- Solaufein Flirt Pack

- Oversight

- The Unusual Oddities Shop

- RPG Dungeon Item Pack

- Improved Asylum

- Allison NPC

- Tactics

- NPC Flirt Packs

- NPC Banter Packs

- Unfinished Business

- Dungeon-Be-Gone

- G3 Fix Pack

- Yasraena

- G3 Tweaks

 

If you find a mod that does conflict with Kitanya, please send an e-mail to:  rastor@rpgdungeon.net with "Kitanya" in the subject line and I will either fix the problem or include a warning with the mod.  You may also post your report on the RPG Dungeon forums.  Also please post any bugs that you encounter with the mod so that I may continue improving the mod.

 

 

Known Issues

 

None (knock on wood).  Send me an e-mail or post on the RPG Dungeon forums (http://forums.rpgdungeon.net) if you find a significant bug.
